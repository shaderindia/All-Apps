1. What is Constructive Solid Geometry (CSG)?

Constructive Solid Geometry (CSG) is a powerful modeling technique that allows you to create complex shapes from basic geometric primitives. By applying boolean operations (Union, Subtract, Intersect), you can hollow out cylinders, cut keyways into shafts, and build custom brackets without the need for manual vertex editing.

2. How to Use the Parametric Modeler

  1. Insert a Primitive: Use the "Add Features" toolbar group to place a Box, Cylinder, Sphere, or Text primitive in your workspace.
  2. Set Parameters: Select the feature in the Feature Tree (left panel) and modify its dimensions, rotation, or coordinates in the Properties panel (right panel).
  3. Apply Boolean Operations: Move one shape inside another and change its operation mode to "Subtract" to engrave text or cut holes.
  4. Inspect and Export: Switch to wireframe mode to review internal structure, then click "STL Out" to save your model to your computer.

3. Tips for 3D Printing & CNC Machining

  • File Scale: All dimensions in the modeler are measured in millimeters, which is the standard unit for CAM software and slicers.
  • Resolution: Keep cylinder facet divisions around 32-64 for smooth rounds without overloading browser memory.
  • Local Backups: Use the "Save" button to download your work as a lightweight `.json` design file, which you can load back into the editor later.
