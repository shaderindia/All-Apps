SHADER7 Circle Geometry Tool

Circle Angle Length Calculator

Enter circle diameter and angles. The app calculates radius, horizontal length and vertical height, then generates a live drawing.

Input

Use comma separated angles like 22.5, 45, 67.5.

Radius will be diameter ÷ 2.
Allowed angle range: 0° to 90°.
Formula
Radius = Diameter ÷ 2
Horizontal length = Radius × cos(angle)
Vertical height = Radius × sin(angle)

Output

The two straight line values are horizontal length and vertical height.
