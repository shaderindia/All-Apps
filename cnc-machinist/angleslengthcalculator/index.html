Angle Length Calculator

Enter circle diameter and angles. Get the radius, horizontal length, vertical height, and a generated circle diagram instantly.

SHADER7 › CNC Machinist › Angle Length Calculator

Input Values

Use one angle or multiple comma-separated angles like 22.5, 45, 67.5.

Radius = diameter ÷ 2.
Allowed angle range: 0° to 90°.
Formula
Radius = Diameter ÷ 2
Horizontal length = Radius × cos(angle)
Vertical height = Radius × sin(angle)
Slanted line / hypotenuse = Radius
For your example diameter 357, radius is 178.5. At 45°, horizontal and vertical values are both equal.

Calculated Output

The two straight line values are horizontal length and vertical height.
