Quick FANUC Reference

Essential G-Codes

  • G00 - Rapid positioning
  • G01 - Linear interpolation (requires F)
  • G02/G03 - Circular interpolation (CW/CCW)
  • G04 - Dwell (e.g., G04 X1.0)
  • G17/G18/G19 - Plane selection
  • G20/G21 - Inches/Metric
  • G28 - Return to reference
  • G40/G41/G42 - Cutter comp cancel/left/right
  • G43/G49 - Tool length comp apply/cancel
  • G54-G59 - Work offsets

Common M-Codes

  • M00 - Program stop
  • M03/M04 - Spindle on (CW/CCW)
  • M05 - Spindle stop
  • M06 - Tool change
  • M08/M09 - Coolant on/off
  • M30 - Program end & reset
